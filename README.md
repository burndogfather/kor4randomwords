# kor4randomwords  
고유명사로 이뤄진 한글 랜덤 단어 생성기  


맑은하늘, 늦은호송, 슬픈저주와 같이 한국어로 구성된 4글자 고유명사를 생성할 수 있습니다.  

4자리 숫자 (10,000개)보다 더 다양한 경우의 수를 지원합니다.  
(80개 형용사 + 2059개 명사 = 총 164,720개의 경우의 수)
